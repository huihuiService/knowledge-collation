# 8种基础数据类型
+ boolean
+ byte 1个字节
+ char 2个字节
+ short 2个字节
+ int 4个字节
+ long 8个字节
+ float 4个字节
+ double 8个字节

# 分类基础数据类型
+ 数值型
    + 整数型
        + byte、short、int、long
    + 浮点型
        + float、double
+ 字符型
    + char
+ 布尔型
    + boolean
# 相关整理
